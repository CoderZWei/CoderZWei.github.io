Java中extend与implements的区别

Posted on 2019-03-13 | In 技术

最近在重新学习Android，extends与implements是十分常见的语法，这里就浅谈一下它们之间的区别，作为笔记。

extends(继承类)

implements(实现接口)

区别：

1）

​ Java中继承类只能继承一个，而实现接口则可以实现多个，如A extends B implements C,D,E

extends继承父类时，父类不能是声明final的或者类定义为abstract，其余的都可以继承

2）

​ 若有父类A，接口B(interface)，自定义类C继承自A类，实现B接口，则C可使用父类A中的方法，若A中有虚函数，则必须实现;同样由于B是接口，则必须实现B中的虚函数。

因此extends是继承功能，而implements是扩展功能
