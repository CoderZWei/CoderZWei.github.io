Linux开发板网络设置

Posted on 2019-03-09

实验室的项目用到的开发板是基于Linux的，为了将其联网，折腾了好一阵，以下为两种联网方式的介绍。

基本环境介绍：网线接口ip为10.0.0.60，网关为：10.0.0.21，默认DNS地址为202.112.128.50，备用DNS地址为202.112.128.51

1、

第一种方法是将板子通过网线直接连到网口上：

​ 1）

​ vim /etc/network/interfaces

​ 编辑内容为：

​ auto lo
​ iface lo inet loopback
​ auto eth0
​ iface eth0 inet static
​ address 10.0.0.60
​ gateway 10.0.0.21
​ netmask 255.255.255.0

​ 2）

​ vim /etc/resolv.conf （重启电脑后会失效，解决办法是 vim /etc/resolvconf/resolv.conf.d/base）

​ 编辑内容为：

​ nameserver 202.112.128.50
​ nameserver 202.112.128.51
​ route add default gw 202.112.128.50 #添加默认网关

​ 3）重启网络配置

​ sudo /etc/init.d/networking restart

2、第二种方法是将板子和电脑连接，然后通过电脑连接外网

​ 1)

​ 电脑为Windows系统，连接方式为无线连接，打开控制面板->网络和Internet->网络和共享中心->更改适配器设置，选择WLAN连接->属性->共享，勾选”允许其他网络用户通过此计算机的Internet连接来连接”，然后选择和板子连接的接口，点击”确定”。注意此时会默认给这个接口分配IP：192.168.137.1。

​ 接着打开板子，进入系统

​ 2)vim /etc/network/interfaces

​ 编辑内容为：

​ auto lo
​ iface lo inet loopback
​ auto eth0
​ iface eth0 inet static
​ address 192.168.137.X #随便设一个不为1的数
​ gateway 192.168.137.1 #即把电脑接口IP设为网关
​ netmask 255.255.255.0

​ 3）

​ vim /etc/resolv.conf （重启电脑后会失效，解决办法是 vim /etc/resolvconf/resolv.conf.d/base）

​ 编辑内容为：

​ nameserver 202.112.128.50
​ nameserver 202.112.128.51
​ route add default gw 202.112.128.50 #添加默认网关

​ 4）重启网络配置

​ sudo /etc/init.d/networking restart
