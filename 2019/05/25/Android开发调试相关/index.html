Android开发调试相关

Posted on 2019-05-25

快捷键：

1）Ctrl+F12：查看类中的所有变量、方法、内部类、内部接口。

2）Ctrl+Tab：切换面板或文件，类似于Windows下的Alt+Tab

3）Ctrl + N：快速查找类

4）Ctrl+R：在当前文件中替换输入的内容

5）Ctrl+Shift+R：全局替换

6）Ctrl+Shift+F 或 Double Shift：全局搜索。Double Shift 搜索的范围更广，支持符号检索，除了Ctrl + N、Ctrl + Shift + N的检索内容外，还可以搜索到变量、资源id等

7）Shift+F6：快速重命名。选中某个类、变量、资源id等之后，可以快速重命名，只要改动一个位置，其它地方也都会自动全部重命名。

8）Alt+F7：快速查找某个类、方法、变量、资源id被调用的地方

9）Ctrl+D：快速复制行

10）Alt+Insert：快速插入代码。可以快速生成构造方法、Getter/Setter方法等

11）Alt+Enter：快速修复错误

12）Ctrl+Alt+Space：类名或接口名提示。输入一个不完整的类名或者接口名，按Ctrl + Alt + Space，会给出完整类名或接口名的提示

13）Ctrl + Shift + I：快速查看某个方法、类、接口的内容。将鼠标光标定位到某个方法、类名、接口名，然后按Ctrl + Shift + I，将会在当前光标位置显示改方法、类、接口的内容

14）Ctrl+P：显示参数信息

15）Ctrl+/：单行注释

16）Ctrl+Shift+/：多行注释

17）Ctrl+Alt+O：优化导入的类和包

18）Ctrl+F12：显示当前文件的结构

19）Ctrl+H：显示类结构图
