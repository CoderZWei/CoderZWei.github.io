198、打家劫舍

解法：

动态规划：dp[i] = max(num[i] + dp[i - 2], dp[i - 1])

dp[i] 表示 [0, i] 区间可以抢夺的最大值，对当前i来说，有抢和不抢两种互斥的选择，不抢即为 dp[i-1]等价于去掉 nums[i] 只抢 [0, i-1] 区间最大值），抢即为 dp[i-2] + nums[i]（等价于去掉 nums[i-1]）.

注意： dp[1]=max(nums[0],nums[1])

236、二叉树的最近公共祖先

解法：

当root非空时，不断从其左右子树搜索：

1）如果一个节点在左子树，一个在右子树，则当前节点为最小公共祖先

2）如果两个节点都在左子树，则对左子树递归搜索

3）如果两个节点都在右子树，则对右子树递归搜索

280、Wiggle Sort 摆动排序

解法：

先给数组排个序，然后我们只要每次把第三个数和第二个数调换个位置，第五个数和第四个数调换个位置，以此类推直至数组末尾，这样我们就能完成摆动排序了

324、 摆动排序 II

解法：

先给数组排序，然后在做调整。调整的方法是找到数组的中间的数，相当于把有序数组从中间分成两部分，然后从前半段的末尾取一个，在从后半的末尾去一个，这样保证了第一个数小于第二个数，然后从前半段取倒数第二个，从后半段取倒数第二个，这保证了第二个数大于第三个数，且第三个数小于第四个数，以此类推直至都取完。

注： 边界不太好控制

376、摆动序列

解法：https://www.cnblogs.com/grandyang/p/5697621.html

动态规划：维护两个dp数组p和q，其中p[i]表示到i位置时首差值为正的摆动子序列的最大长度，q[i]表示到i位置时首差值为负的摆动子序列的最大长度。我们从i=1开始遍历数组，然后对于每个遍历到的数字，再从开头位置遍历到这个数字，然后比较nums[i]和nums[j]，分别更新对应的位置
