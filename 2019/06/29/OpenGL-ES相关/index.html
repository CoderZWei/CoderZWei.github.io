Posted on 2019-06-29

记录OpenGL ES的相关知识点

1、GLSurfaceView的渲染器(Renderer需要实现三个接口)：

1）public void onSurfaceCreated(GL10 gl10, EGLConfig eglConfig) {}

2）public void onSurfaceChanged(GL10 gl10, int width, int height) {

3） public void onDrawFrame(GL10 gl10) {)

​ 在此方法中调用 GLES20.glClear(GLES20.GL_DEPTH_BUFFER_BIT | GLES20.GL_COLOR_BUFFER_BIT); 清空屏幕

GL10是OpenGL ES1.0的api遗留下来的，如果要编写使用OpenGL ES1.0 的渲染器，就要使用这个渲染器；对于OpenGL ES2.0，GLES20类提供了静态类方法来存取

GLSurfaceView使用：

//这是OpenGL版本 必需

mGLSurfaceView.setEGLContextClientVersion(2);

//设置渲染器

mGLSurfaceView.setRenderer(new MyRenderer());

//设置渲染模式为连续模式(会以60fps的速度刷新)

mGLSurfaceView.setRenderMode(GLSurfaceView.RENDERMODE_CONTINUOUSLY);
