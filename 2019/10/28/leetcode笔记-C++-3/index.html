374、猜数字大小

474、一和零

解法：动态规划，注意是从右下角到左上(为什么？)

dp[i][j]=max(dp[i][j], dp[i - zeros][j - ones] + 1) ,dp[i][j]表示有i个0和j个1时能组成的最多字符串的个数，而对于当前遍历到的字符串，我们统计出其中0和1的个数为zeros和ones。
