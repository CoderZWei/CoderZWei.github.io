377、组合总和IV

解法：

1）动态规划，dp[i]表示目标数为i的解的个数，然后从1遍历到target，对于每一个数i，遍历 nums 数组，如果 i>=x, dp[i] += dp[i - x]。

242、有效的字母异位词

解法：

先用一个map遍历存储字符出现的个数，然后遍历另一个，若遍历过程中小于0，则直接返回false

386、字典序排数

解法：

https://www.cnblogs.com/grandyang/p/5798275.html

按个位数遍历，在遍历下一个个位数之前，先遍历十位数，十位数的高位为之前的个位数，只要这个多位数并没有超过n，就可以一直往后遍历，如果超过了，我们除以10，然后再加1，如果加1后末尾形成了很多0，那么我们要用个while循环把0都去掉，然后继续运算
