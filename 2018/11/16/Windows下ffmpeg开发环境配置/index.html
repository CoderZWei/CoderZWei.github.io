Windows下ffmpeg开发环境配置

Posted on 2018-11-16 | In 技术

系统环境为Windows，开发环境为visual studio2015

1、首先下载ffmpeg，从官网下载即可，总共分为三种类型：

1）static：提供了静态版本的ffpmeg工具(三个exe可执行文件)，将依赖的库生成在了最终的可执行文件中

2）share：该版本的工具包括可执行文件和dll动态库，程序运行过程必须依赖于提供的dll文件

3）dev：提供了库的头文件和dll的引导库

2、将share版本中的bin目录路径加入path环境变量可在命令行中调用ffmpeg，注意其中要有dll

3、注：此步骤使用的是dev版本中的文件

新建visual studio 空项目，在属性管理器(可在 视图->其他窗口->属性管理器 打开)中的对应版本中右键->添加新项目属性表 (添加所需的ffmpeg文件)

1）在 C++ -> 常规 -> 附加包含目录 中添加include目录

2）在 链接器 -> 常规 -> 附加库目录 中添加lib目录

3）在 链接器 -> 输入 -> 附加依赖图 中添加lib目录下的lib库

以上可以生成配置文件，之后新建项目时引入此配置文件即可

4、写代码测试
