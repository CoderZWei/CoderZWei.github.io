github搭建博客教程

Posted on 2017-11-05 | In 技术

之前都是在CSDN写博客，但是其排版实在是不美观且不方便，在GitHub上拥有一个属于自己的博客也是蛮有逼格的，通过hexo框架，使用markdown语法直接写博客也十分方便。看了网上的一些教程，自己实践搭建的时候踩了很多坑，以此博客来记录搭建博客的过程。

首先新建一个仓库，名为 github名.github.io ，如下图所示：

​ 这时便可在 https://github名.github.io/ 访问自己的博客，还可以使用github提供的主题，如果觉得那些主题不太美观，操作不太方便，可以使用第三方框架，这里我使用的是十分流行的hexo框架，其操作十分方便，新建文章只需几行命令，还有很多样式可供选择。

​ 欲使用hexo，需安装hexo、node.js、Git

​ 其中node.js和Git到官网下载软件然后点击安装一路next就可以，hexo在命令行中输入：npm install nexo-cli -g安装。

接着熟悉一下hexo的几个基本操作

​ 接着在本地新建一个文件夹，文件夹名为刚刚新建的仓库名，进入文件夹右键“Git Bash Here”，输入”hexo init”初始化网站

​ 打开目录下的_config.yml文件，将theme标签内容改为”next“(我这里用的是next主题，也可以使用其它主题或默认配置)

​ 将deploy标签改为如下所示：

​ 检查SSH keys的设置：输入”ssh -T git@github.com”查看是否连接成功，若失败可查看这个教程进行配置

​ 新建一个博客：hexo n “test”，输入”hexo g”重新生成，输入”hexo s”在本地查看效果

​ 最后输入：hexo d 提交到github合并，在浏览器中输入之前所说的地址便可以浏览博客
